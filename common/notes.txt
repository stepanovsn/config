============= Archives =============

tar:
tar cvzf name.tar.gz src_dir
tar tvzf name.tar.gz
tar xvzf name.tar.gz -C dest_dir

zip:
zip name.zip -r src_dir
unzip -l name.zip
unzip name.zip -d dest_dir

rar:
rar a -r name.rar src_dir
unrar l
unrar e name.rar dest_dir

============= System info =============

Various system info:
lsblk -f
lscpu
lspci -v
lsusb -v

Collect all system info:
inxi -Fx
lshw -short
hwinfo --short

Get system info from BIOS:
dmidecode

============= Other =============

stat:
Info about file status: stat filename.txt
Info about file system status: stat -f filename.txt

fzf:
For autocompletion use: some_command **<Tab>

mmv:
Rename multiple files extensions: mmv '*.cpp' '#1.c'

chmod:
Categories: u - user, g - group, o - other.
For recursive use: -R
Grant all rights to all users: chmod ugo+rwx filename.txt

chown:
Change ownership: chown user file.txt
Change ownership and group: chown user:user file.txt
Change ownership(recursively): chown -R user dir/

Determine file type:
file filename
